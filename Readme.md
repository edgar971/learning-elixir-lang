## Useful commands

- Running Docker container: 

        docker run -it -v $(pwd)/:/app --rm elixir


- Reload Module

        r(ModulePlayground)

- Load Files: 

        load_file("file_name.exs")

- Mix is a CLI for Elixir. 

        mix help

- Run Test

        mix Test

- Install Packages

        mix deps.get

- Run the Application 

        mix run --no-halt
